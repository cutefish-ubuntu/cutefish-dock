# cutefish-dock
Desktop Taskbar

You also need [`fishui`](https://github.com/cutefishos/fishui) and [`libcutefish`](https://github.com/cutefishos/libcutefish).

## Build and Install

```
git clone https://github.com/cutefishos/dock.git
mkdir ~/dock/build $$ cd ~/dock/build
cmake ..
make
sudo make install
```
